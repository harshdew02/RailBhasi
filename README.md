# RailBhasi

This is our main repo.

## Run Locally

Clone the project

```bash
  git clone https://github.com/harshdew02/RailBhasi.git
```

Go to the project directory

```bash
  cd RailBhasi
```

Install dependencies

```bash
  npm install
```

Start the server

```For CLI
  npm run dev
```

```For Expo run
  npx expo start --offline
```
